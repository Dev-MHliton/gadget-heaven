## 📖 Overview

**Gadget Heaven** is a frontend e-commerce application built using modern web technologies.  
It allows users to browse gadgets, view details, manage cart items, and enjoy a clean shopping experience.

---

## ✨ Key Features

### 🛍️ Product Experience
- Browse a wide range of gadgets
- View detailed product information
- Organized product categories

### 🛒 Cart System
- Add/remove products from cart
- Persistent cart storage using `localforage`
- Smooth cart interaction experience

### 🔍 Smart Filtering
- Search and sort products
- Match-based filtering using `match-sorter`
- Easy navigation across products

### ⚡ UI & UX
- Fully responsive design
- Modern UI using Tailwind + DaisyUI
- Fast and smooth navigation

---

## 🛠️ Tech Stack

### Frontend
- React 19
- React Router DOM
- React Icons
- React Tabs
- React Helmet

### Styling
- Tailwind CSS 4
- DaisyUI

### Utilities
- localforage (offline storage)
- match-sorter (search filtering)
- sort-by (sorting logic)

### Build Tool
- Vite 6

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Dev-MHliton/gadget-heaven.git
```

### Navigate to the Project Directory

```bash
cd e-ticket-system
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
